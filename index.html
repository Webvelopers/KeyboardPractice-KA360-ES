Como desbloqueas el avance

  • 95%precision minima.

Como funciona

  1. Empezar entrenamiento: activa el campo de escritura, reinicia el temporizador y bloquea cambios de leccion.
  2. Finalizar intento: guarda metricas, evalua precision, PPM, errores y tiempo maximo objetivo.
  3. Repetir: borra el intento actual o prepara una nueva repeticion sin cambiar el progreso aprobado.
  4. Siguiente leccion: aparece habilitado solo cuando la proxima leccion fue desbloqueada.
  5. Atajos: usa Alt+K en Windows/Linux o ⌘+K en macOS para iniciar, Esc para finalizar, Alt+H/⌘+H para avanzar y Enter/⏎ para cerrar mensajes emergentes.
